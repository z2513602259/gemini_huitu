# UI 重构计划：打造美观高级的 AI 绘图工作台

本计划旨在将现有的 AI 绘图应用重构为具有现代感、高级感和科技感的界面，采用 **Glassmorphism (毛玻璃)** 风格与 **深色模式** 相结合的设计语言。

## 1. 技术栈升级
- **引入 Tailwind CSS**：安装并配置 Tailwind CSS，利用其强大的工具类快速构建响应式和现代化 UI。
- **配置 PostCSS**：确保 Tailwind 正常工作。

## 2. 设计风格定义 (Glassmorphism + Dark Mode)
- **配色方案**：
  - **背景**：深邃的午夜蓝/黑 (`Slate 950`) 配合动态流转的极光渐变（紫/绿），营造神秘且富有创造力的氛围。
  - **面板**：高透明度的磨砂玻璃效果 (`bg-white/5 backdrop-blur-xl border-white/10`)，让背景光晕隐约透出，增加层次感。
  - **文字**：高对比度的白色/灰白色 (`Slate 50`/`Slate 300`)，确保可读性。
  - **强调色**：电光紫 (`Violet 500`) 和 荧光绿 (`Emerald 400`) 的渐变，用于按钮和关键交互元素。
- **视觉元素**：
  - **圆角**：统一使用较大的圆角 (`rounded-2xl` 或 `rounded-3xl`)，显得更柔和现代。
  - **阴影**：使用彩色的弥散阴影 (`shadow-violet-500/20`) 代替传统的黑色投影，增加发光感。
  - **边框**：极细的半透明边框 (`border border-white/10`)，增强精致感。

## 3. 布局重构 (Layout Redesign)
- **整体结构**：采用 **Split View (分栏布局)**。
  - **左侧控制区 (Sidebar/Panel)**：宽度固定或响应式调整，包含所有参数设置（提示词、尺寸、上传、工作流选择）。
  - **右侧预览区 (Canvas)**：占据剩余空间，居中展示生成的图片，提供沉浸式的查看体验。底部或顶部悬浮展示历史记录缩略图。
- **导航栏**：简化为顶部透明浮动栏，包含 Logo、模式切换（生图/工作流）和设置入口。

## 4. 组件优化
- **输入框/Textarea**：移除默认样式，改为半透明填充 + 底部发光边框，聚焦时产生光晕。
- **按钮**：
  - **主按钮 (Generate)**：设计为醒目的渐变色胶囊按钮，带有微弱的呼吸动画。
  - **次要按钮**：使用 Ghost 风格，Hover 时背景轻微变亮。
- **卡片 (工作流选择)**：设计为可交互的玻璃卡片，Hover 时上浮并高亮边框。
- **图片上传区**：设计为极简的虚线框，支持拖拽，上传后以精美的网格展示预览。
- **Loading 状态**：保留并优化现有的陀螺动画，或添加更具科技感的进度条。

## 5. 实施步骤
1.  **环境配置**：安装 Tailwind CSS 及相关依赖，生成配置文件。
2.  **样式迁移**：将 `src/styles.css` 中的核心动画和变量迁移到 Tailwind 配置或新的 CSS 层，删除旧的全局样式。
3.  **组件重构**：
    - 修改 `App.tsx` 布局结构。
    - 重构 `GenerateButton`、`HistoryView` 等组件，替换 `className` 为 Tailwind 工具类。
    - 优化表单元素和交互反馈。
4.  **细节打磨**：调整间距、字体大小、颜色对比度，确保视觉平衡。
5.  **验证**：确保所有功能（生图、历史记录、下载、拖拽）在 UI 重构后依然正常工作。
